# Installation

Use the existing lockfile and scripts in `package.json`.

<Tabs items={['yarn dev', 'yarn build', 'yarn start']}>
  <Tabs.Tab>

  Start the development server:

  ```bash
  yarn dev
  ```

  </Tabs.Tab>
  <Tabs.Tab>

  Build the production app:

  ```bash
  yarn build
  ```

  </Tabs.Tab>
  <Tabs.Tab>

  Serve the production build:

  ```bash
  yarn start
  ```

  </Tabs.Tab>
</Tabs>

<Steps>
## Install dependencies

This example already includes a `yarn.lock`, so stick with Yarn for local development.

## Start the app

Run `yarn dev` and open the local URL printed by Next.js.

## Add content

Create more `.mdx` files under `content/` and they will be picked up by the Nextra page map.
</Steps>
