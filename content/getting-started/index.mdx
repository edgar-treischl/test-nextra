# Getting started

This repository keeps its docs pages in `content/` and lets Nextra load them through a catch-all route in `app/[[...mdxPath]]/page.jsx`.

## Request flow

1. Next.js matches the incoming path to the catch-all route.
2. `importPage(...)` resolves the corresponding MDX file from `content/`.
3. The docs theme wrapper renders the page with the sidebar, table of contents, and page metadata.
