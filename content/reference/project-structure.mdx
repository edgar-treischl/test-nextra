# Project structure

## `next.config.mjs`

Registers the Nextra plugin with Next.js.

## `app/layout.jsx`

Defines the shared docs shell: banner, navbar, footer, head tags, and page map.

## `app/[[...mdxPath]]/page.jsx`

Loads MDX pages from `content/`, generates metadata, and renders them through the docs theme wrapper.

## `mdx-components.js`

Merges the docs theme MDX components with any project-level overrides.

## `content/`

Holds the actual documentation pages that power the site.
